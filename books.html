Field Notes Archive

FALSE HYPOTHESIS

WebRTC Signaling Fallback

Assumed public PeerJS relays were reliable enough for initial handshake. They are not. Introduced race conditions in identity creation. Rewriting signaling protocol to require explicit STUN fallback.

ABANDONED THREAD

Cloud Backup Service

Considered adding optional cloud backup for Zayvora profiles. Abandoned. Too slippery a slope. Enforces the constraint: the user must physically manage their `.zay` cold storage.

MAINTENANCE

OPFS Quota Limits

Hit Origin Private File System limits during intensive PDF parsing. Implemented chunked eviction strategy for temporary cache blocks.

RETURNED PROBLEM

Async UI Freezes

Heavy crypto operations block the main thread. Returning to the Web Worker implementation designed in Vol 1 to isolate SubtleCrypto.

# Zhuangzi

**The Butterfly Dreamer**

| | |
|---|---|
| Era | 369–286 BC |
| Origin | Song, China |
| Field | Philosophical Taoism |

"Am I a man dreaming I'm a butterfly, or a butterfly dreaming I'm a man?" Zhuangzi's radical relativism and playful paradoxes challenged all fixed perspectives on reality.

# Pythagoras

**The Harmonic Mystic**

| | |
|---|---|
| Era | 570–495 BC |
| Origin | Samos, Greece |
| Field | Mathematics & Mysticism |

Pythagoras founded a philosophical and religious movement. He discovered the mathematical ratios underlying musical harmony and believed that numbers are the ultimate reality behind all things.

# Georg Wilhelm Friedrich Hegel

**The Dialectician**

| | |
|---|---|
| Era | 1770–1831 |
| Origin | Stuttgart, Germany |
| Field | Absolute Idealism |

Hegel proposed that history unfolds through dialectical conflict—thesis, antithesis, synthesis. His system of Absolute Idealism influenced Marx, existentialism, and modern political theory.

# Socrates

**The Gadfly of Athens**

| | |
|---|---|
| Era | 470–399 BC |
| Origin | Athens, Greece |
| Field | Ethics & Dialectic |

Socrates never wrote a word, yet transformed philosophy forever. Through relentless questioning—the Socratic Method—he exposed ignorance and pursued virtue, ultimately choosing death over abandoning his principles.

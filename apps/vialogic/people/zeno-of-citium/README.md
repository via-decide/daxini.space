# Zeno of Citium

**The Stoic Founder**

| | |
|---|---|
| Era | 334–262 BC |
| Origin | Citium, Cyprus |
| Field | Stoicism |

Zeno founded Stoicism, teaching that virtue is the sole good and that we should accept what we cannot control with equanimity. His philosophy shaped Roman leaders and echoes in modern cognitive therapy.

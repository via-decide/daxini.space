# Rumi

**The Whirling Poet**

| | |
|---|---|
| Era | 1207–1273 |
| Origin | Balkh, Afghanistan |
| Field | Sufi Mysticism |

Rumi's poetry speaks of divine love, unity, and the journey of the soul. His Masnavi is called "the Quran in Persian." He remains the best-selling poet in America eight centuries later.

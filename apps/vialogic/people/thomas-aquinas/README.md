# Thomas Aquinas

**The Angelic Doctor**

| | |
|---|---|
| Era | 1225–1274 |
| Origin | Roccasecca, Italy |
| Field | Scholasticism |

Aquinas synthesized Aristotelian philosophy with Christian theology. His Five Ways arguments for the existence of God and his natural law ethics remain central to Catholic thought.

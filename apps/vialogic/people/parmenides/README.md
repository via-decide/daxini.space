# Parmenides

**The Prophet of Being**

| | |
|---|---|
| Era | 515–450 BC |
| Origin | Elea, Italy |
| Field | Ontology |

Parmenides argued that change is an illusion and that reality is one, eternal, and unchanging. His poem "On Nature" laid the groundwork for metaphysics and the study of Being.

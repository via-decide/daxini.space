# Søren Kierkegaard

**The Father of Existentialism**

| | |
|---|---|
| Era | 1813–1855 |
| Origin | Copenhagen, Denmark |
| Field | Existentialism |

Kierkegaard insisted that truth is subjective and that authentic existence requires a "leap of faith." He rejected Hegel's systems in favor of individual passion and choice.
